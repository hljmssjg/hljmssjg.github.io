3. JRE和JDK

3.1 JRE

JRE（Java Runtime Environment）是Java程序的运行时环境，包含JVM和运行时所需要的核心类库。JRE可以实现跨平台。可以实现Java运行。

3.2 JDK（最重要的）

是Java程序开发工具包，包含JRE和开发人员使用工具。（JDK是最牛b的）
